# CustomerView-Kotlin

This project is implemented by kotlin based on AndroidX and shows you some interesting customer views. 

![](./CustomerView.gif)

It includes:

6. [Bezier Wave View](./app/src/main/java/com/jay/kotlin/customerview/customer/BezierWaveView.kt)
